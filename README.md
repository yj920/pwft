# PHP Web File Tool

## Getting Started

### Prerequisites 

```
$ sudo apt-get install git
$ sudo apt-get install php
```

### Installation 

```
$ git clone https://github.com/yj920/pwft.git
$ cd pwft
$ mkdir file
$ php -S localhost:8080
$ firefox localhost:8080
```
